# Claude Code — Global Preferences

## Shell Preferences

- **Always use `trash` instead of `rm`** — `trash` is the macOS built-in that moves files to Finder's Trash (reversible), rather than permanently deleting them.
- **Always use `bb` CLI for Bitbucket** — only fall back to browser automation for features the CLI doesn't support. If the CLI doesn't work (not installed, auth issues, etc.), abort and ask the user for help/setup.
